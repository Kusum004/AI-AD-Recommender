# AI-AD-Recommender
# 🎯 Multi-Modal Personalized Ad Recommender

An AI-driven e-commerce marketing dashboard that combines visual features, user behavior sequences, and anomaly detection to optimize ad targeting and reduce churn.

## 🚀 Key Features (MSc AI/ML Syllabus Units 3, 4, 5)
- **CNN Visual Feature Extraction (Unit 3):** Analyzes product images to understand style and category "vibes."
- **LSTM Sequence Prediction (Unit 4):** Processes user clickstreams to forecast the next likely purchase category.
- **Autoencoder Anomaly Detection (Unit 5):** Filters out bot traffic and "fake clicks" to protect marketing spend.
- **Interactive Simulation:** A Streamlit dashboard allowing marketers to adjust budgets and visualize "Retention Lift."

## 🛠️ Tech Stack
- **Language:** Python
- **Deep Learning:** TensorFlow / Keras
- **Web App:** Streamlit
- **Visualization:** Plotly & Matplotlib
- **Dataset:** UCI Online Retail II

## 📈 Business Impact
This tool moves beyond standard recommendation systems by introducing a security layer (Autoencoders) and visual understanding (CNNs), directly impacting **Customer Retention** and **Ad Spend Efficiency**.
